About the Engine

What is Aletheia?

Aletheia (Greek: ἀλήθεια, meaning Truth) is an AI-powered fake news detection system that uses a multi-layer decision pipeline to classify news as REAL, FAKE, or UNCERTAIN. It combines deep learning, machine learning, and rule-based credibility analysis to provide reliable, transparent predictions with confidence scores and keyword explanations.

How It Works — 4-Layer Pipeline

1
Credibility Signal Engine

Scans for sensational language, fake news buzzwords ("miracle cure", "they don't want you to know"), and credibility markers (Reuters, peer-reviewed, official statement). Fast O(n) rule scan.

2
RoBERTa Deep Learning (BERT API)

Uses the hamzab/roberta-fake-news-classification model via the Hugging Face Router to understand semantic context, tone, and structure. Handles complex nuanced text that simple rules cannot classify.

3
Uncertainty Detection

Flags texts with heavy hedging language ("may suggest", "allegedly", "reportedly", "under review") as UNCERTAIN to avoid overconfident wrong predictions.

4
Consensus Engine + Override Logic

Combines BERT output and rule scores. If BERT says REAL but fake buzzwords overwhelm the text, it overrides to FAKE. If BERT fails, rule-based engine takes over.
